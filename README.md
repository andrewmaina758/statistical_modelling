# Statistical Modelling

This repository showcases a collection of statistical modelling projects I have completed, focusing on both classical and probabilistic approaches. It highlights practical, real‑world applications and experiments across a range of model families, with an emphasis on clear methodology, interpretation, and reproducibility.

## Highlights

- **Linear regression** models for continuous outcomes
- **Logistic regression** for binary classification
- **Linear Discriminant Analysis (LDA)** and **Quadratic Discriminant Analysis (QDA)** for supervised classification
- **Multinomial regression** for multi‑class classification
- **Poisson regression** for count data

## What you’ll find here

- End‑to‑end modelling workflows (data preparation → modelling → evaluation)
- Model comparisons and performance analysis
- Feature engineering and diagnostics
- Interpretable results and visualizations

## Model families covered (non‑exhaustive)

- Linear models (OLS, Ridge/Lasso as applicable)
- Generalized Linear Models (GLMs)
- Discriminant analysis
- Count models (Poisson)

## Purpose

The goal of this repository is to document and share my statistical modelling work, serve as a reference for applied projects, and demonstrate practical usage of foundational models across diverse datasets.
